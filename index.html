FriendZone

Friendzone was built with gamers in mind. With this social medial style application, we allow gamers to create a profile with their "gamer-tags" from any platform, search/follow all of the games they play, and follow their friends. Once a user follows a specific game, they can enter a game specific chat to meet up with other people. After game play, users are able to up/down vote other gamers, and "gamer rep" will be reflected publicly. Built with a Node.js backend, mongoDB database, Auth0 for user management and a Vue.js frontend.

Kanban

With Kanban, a peer and I build a visually pleasing task manager application. A user is able to create a task board and can then create tasks specific to that board and lists specific to those tasks. From there users can create comments on those tasks. Boards can be user specific by logging in with Auth0. Built with a Node.js backend, mongoDB database, Auth0 for user management and a Vue.js frontend.

Keepr

Keepr is a full stack social media application that allows users to visually share and discover new interests by posting images or videos to their own collection. Users that are not logged in can see all public posts and can open them for more details. Built with a .Net WebApi implementing the repository pattern to communicate with a MySQL database, Auth0 for user management, and a Vue.js frontend.
